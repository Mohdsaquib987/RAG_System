# RAG System

This is a **Retrieval-Augmented Generation (RAG) project** built for practice and experimentation. It allows querying PDFs  using an AI model with a vector database backend.

---

## Project Structure
RAG_System/
├── backend/ # Backend code & database
│ ├── chroma_db/ # Chroma vector database files
│ ├── qdrant_db/ # Qdrant database files
│ ├── pdfs/ # Sample PDFs
│ ├── rag.py # Local practice file (removed from repo)
│ └── .env.example # Template for environment variables
├── frontend/ # Frontend code
└── .gitignore # Git ignore rules
